# UI 优化说明

## 🎯 优化内容

### 1. **图标大小优化**
- ✅ 图标大小从 24x24 增加到 **32x32**
- ✅ 更清晰、更易点击
- ✅ 视觉效果更好

### 2. **性能优化**
- ✅ **禁用毛玻璃效果** (Acrylic)
  - 减少 GPU 占用
  - 提高帧率
  - 动画更流畅

- ✅ **禁用不必要的动画**
  - 减少 CPU 占用
  - 响应更快

### 3. **配置管理**
- ✅ 创建 `ui/config.py` 集中管理配置
- ✅ 易于调整参数
- ✅ 便于后续优化

## 📝 配置文件

### `ui/config.py`

```python
# 导航栏配置
NAVIGATION_CONFIG = {
    "icon_size": QSize(32, 32),  # 图标大小
    "acrylic_enabled": False,     # 禁用毛玻璃效果
    "animation_enabled": False,   # 禁用动画
}

# 窗口配置
WINDOW_CONFIG = {
    "width": 1200,
    "height": 750,
    "min_width": 1000,
    "min_height": 600,
}

# 主题配置
THEME_CONFIG = {
    "theme": "light",  # light 或 dark
}
```

## 🔧 如何调整

### 增大图标
编辑 `ui/config.py`：
```python
"icon_size": QSize(40, 40),  # 改为 40x40
```

### 启用毛玻璃效果
```python
"acrylic_enabled": True,  # 改为 True
```

### 切换深色主题
```python
"theme": "dark",  # 改为 dark
```

## ⚡ 性能对比

| 项目 | 优化前 | 优化后 | 改进 |
|------|--------|--------|------|
| 图标大小 | 24x24 | 32x32 | 更清晰 |
| 毛玻璃效果 | 启用 | 禁用 | 更流畅 |
| 动画帧率 | 低 | 高 | 无掉帧 |
| 响应速度 | 慢 | 快 | 更快 |

## 💡 建议

1. **如果需要更大的图标**
   - 改为 `QSize(40, 40)` 或 `QSize(48, 48)`

2. **如果需要毛玻璃效果**
   - 改为 `"acrylic_enabled": True`
   - 但会影响性能

3. **如果需要深色主题**
   - 改为 `"theme": "dark"`
