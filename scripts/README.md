# Scripts Directory

This directory contains standalone test and utility scripts for development and debugging.

## Files

- **cnocr_test.py**: Standalone OCR testing script for testing CnOcr recognition on sample images
- **relic_detection_poc.py**: Proof-of-concept script for relic state detection using template matching
- **test_filter.py**: Test script for repository filter functionality

## Usage

These scripts can be run independently for testing specific functionality without launching the full application.

```bash
# Test OCR recognition
python scripts/cnocr_test.py

# Test relic detection
python scripts/relic_detection_poc.py

# Test filter functionality
python scripts/test_filter.py
```
